# Working with repositories

## Cloning a repository to a directory

`git clone repository-url`

### Example

`git clone git@github.com:nghianguyentek/prerequisites.git`

## Changing the URL of a cloned repository

`git remote set-url origin new-url`

### Examples

#### Change the URL of the current repository

`git remote set-url origin git@github.com:nghianguyentek/prerequisites.git`
